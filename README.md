# Neurofive Machine Learning Track - Week 1

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) using the Titanic dataset from Kaggle. The aim is to understand the dataset's structure, identify missing values, classify features, and generate insights before building machine learning models.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Tasks Completed
- Imported required libraries
- Loaded the Titanic dataset
- Explored dataset using `head()`, `tail()`, `sample()`, `info()`, and `describe()`
- Identified rows, columns, and data types
- Checked missing and duplicate values
- Classified numerical and categorical features
- Explored unique values and value counts
- Generated a correlation matrix
- Created basic visualizations
- Wrote observations and conclusions

## Dataset Summary
- Rows: 891
- Columns: 10
- Missing Values: Age (177)
- Numerical Features: 7
- Categorical Features: 3

## Learning Outcomes
This project strengthened my understanding of Exploratory Data Analysis (EDA), data inspection, handling missing values, identifying feature types, and visualizing datasets before machine learning.
